      $ npm run setup

   2. Start development server:
      $ npm run dev

   3. Open your browser:
      → http://localhost:3000

✨ FEATURES INCLUDED:

   ✅ Interactive MapLibre map with location search
   ✅ Calendar date picker for any day of year
   ✅ 5 weather categories (hot, cold, windy, wet, uncomfortable)
   ✅ 20 years of historical data analysis
   ✅ Probability calculations & statistics
   ✅ Climate trend analysis (climate change insights)
   ✅ Interactive charts & visualizations
   ✅ CSV & JSON data export
   ✅ Mobile-responsive design
   ✅ Beautiful, modern UI with Tailwind CSS

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🛠️  TECH STACK:

   Frontend:
   • Next.js 14 (App Router)
   • React 18 with TypeScript
   • Tailwind CSS
   • MapLibre GL
   • Recharts

   Backend:
   • Next.js API Routes
   • Open-Meteo Historical API
   • OpenStreetMap Nominatim
